gCTS Task Dependency Analyzer

Eclipse ADT plugin for detecting cross-task object dependencies in SAP gCTS Transport Requests.

Install in Eclipse

  1. Help → Install New Software → Add
  2. Name: gCTS Analyzer
  3. URL: https://mayur175.github.io/tr-analyser/updatesite
  4. Select gCTS Tools for ADT → Install → Restart
